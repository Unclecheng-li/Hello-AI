# 机器学习

机器学习（Machine Learning）是人工智能的一个子领域，其核心就是让计算机从大量的数据当中自动学习规律，而不是靠人手写规则

传统的编程就是告诉程序“如果 A 就做 B ”，而机器学习则是给程序大量的例子，让它自己总结出“如果 A 就做 B”

机器学习的核心思想是让计算机利用算法和统计模型，从大量历史数据（即“经验”）中自动地“学习”其内在规则，并利用学到的规则对未知的新数据进行预测、分类或决策。

这个过程可以概括为：**收集数据 -> 输入数据 -> 数据预处理 -> 训练和测试模型 -> 模型评估**。

那么机器具体是怎么“学”的？主要有以下三种方式
## 三种主要学习方式

**监督学习：** 给机器提供带标签的数据，告诉机器“这个是猫，那个是狗”，让他学会分类学会预测，其中常见的算法有：`线性递归`、`决策树`、`神经网络`等

**无监督学习：** 只给机器数据，不给机器标签，让机器自己找到对应的规律，比如把用户分为几类人群。常见的算法有：`K-Means 聚类`、`PCA 降维`等

**强化学习：** 机器通过不断试错、获得奖励和惩罚来学习的策略，就像训练狗狗一样，AlphaGo下围棋就是这一套

不管哪种方式，底层都涉及同一套核心概念，接下来我们就来理解它们
## 机器学习的核心概念

- **模型：** 机器学习的整个过程，本质上是在调整一个 **模型** 的参数，让它的输出越来越接近期望答案。模型是“规律”的数学载体，可以是一条直线、一颗决策树，也可以是拥有数十亿参数的神经网络
- **特征：** 模型的输入叫做 **特征** ，是你选择喂给它信息，比如在让它写代码时，你的要求的用什么语言，实现什么功能是特征，比如让它进行画图的时候，关键词就是特征
- **训练：** **训练** 是让模型变好的过程，简单说，就是把大量的数据喂给模型之后，用一个**损失函数**的东西衡量它预测得有多差，然后通过**优化算法**不断调整参数，让损失变得的越来越小
- **过拟合：** 训练结束后，模型在训练数据上可能表示得很完美 -- 但是这远远不够，我们真正在乎的是它面对没有见过的新数据的时候如何表现。这里有一个陷阱叫做 **过拟合**，模型训练的时候可能会记住一些**噪音**和细节，变成了死记硬背，泛化能力很差。解决的办法通常是增加数据量、加入正则化、或者利用 **Dropout** 等技术迫使模型不依赖某一些特征。

搞清楚这些概念之后，还有个经常让人混淆的概念值得说一下：

> 机器学习，深度学习，AI 之间到底是什么关系？

他们三者是一个包含关系：

```plain
人工智能（AI）
  L 机器学习（Machine Learning）
        L 深度学习（Deep Learning）
```

`人工智能` 是最宽泛的概念，凡是让机器表现出智能行为的技术都算。机器学习是其中一条路线，核心是从数据学习。深度学习则是机器学习里的一个子集，用的是多层神经网络——之所以叫"深度"，是因为层数很多，能从原始数据里自动提取越来越抽象的特征，省去了人工设计特征的步骤。

`深度学习` 是近十年 AI 爆发的核心驱动力。图像识别、自然语言处理、语音合成，背后基本都是深度学习。ChatGPT、Gemini 这类大语言模型，本质是在海量文本上训练的超大规模深度神经网络，参数量从几十亿到几千亿不等。

但深度学习不是万能的。它需要大量数据和算力，可解释性差（你很难知道模型为什么做出某个决策），在数据少或规则明确的场景，传统机器学习算法（比如随机森林、SVM）反而更实用、更稳定。

**一句话总结：** 机器学习是用数据驱动的方式，让程序自动归纳规律、做出预测或决策——它改变的不只是工具，而是解决问题的思路本身。
