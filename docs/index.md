# Hello-AI

Hello-AI 是一条给中文学习者准备的 AI / LLM 入门路径。

它不追求“内容堆满”，而是把你真正需要的东西排成顺序：

1. 先知道 AI、LLM、Prompt、RAG、Agent 是什么
2. 再学会怎么用
3. 然后做出第一个小项目
4. 最后开始理解评测、安全和部署

## 你可以从这里开始

- 如果你是新手，先看 [新手起步](preface/start.md)
- 如果你想要路线图，先看 [学习路线](preface/roadmap.md)
- 如果你已经接触过一些概念，直接看 [AI 基础](basics/index.md)

## 当前重点

- AI 基础
- Prompt 基础
- 工具使用
- RAG 入门
- 练习与实验

## 站点原则

- 中文优先
- 路径优先
- 小例子优先
- 可复现优先

## 先别急着做什么

第一版不会一上来就做复杂后台、用户系统或推荐系统。先把静态内容、学习路径和反馈机制跑通。
