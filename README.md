# Hello-AI

Hello-AI 是一个面向中文用户的 AI / LLM 学习入口平台，目标是把基础认知、Prompt、工具、RAG、Agent、Build、Eval 和 Safety 串成一条能走通的学习路径。

## 当前状态

项目处于初始化阶段，已完成站点骨架与第一版导航。

## 本地预览

```bash
pip install -r requirements.txt
mkdocs serve
```

然后访问 `http://127.0.0.1:8000/`。

## 目录说明

- `docs/`：正文内容
- `assets/`：图片与附件
- `overrides/`：主题覆盖
- `scripts/`：辅助脚本
- `.github/workflows/`：CI/CD
